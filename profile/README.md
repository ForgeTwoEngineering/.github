# Forge Two Engineering

**Two high school engineers building hardware that moves, flies, and thinks.**

We're Ivan Martinez and Oskar Trujillo — sophomores in Tucson, Arizona. Ivan handles AI and software (Python, Claude API, OpenCV, computer vision). Oskar handles hardware and fabrication (3D printing, circuit wiring, motor control, power systems).

We build things, we finish them, and we document everything.
